Why

This is my website in which I have spoken about all my life experiences, achievements, projects, research work, and publications. Since I love web designing, I wanted to build a website from scratch and decided not to use any frameworks (website generator). Instead, I started using HTML 5, Bootstrap 4, CSS 4, and Javascript. By doing this helped me in mastering my programming skills and thinking out of the box. Sometimes I had to critique my own work. Because designing ain’t easy. Therefore, I built this website from scratch.

When

I started building this website on April 13, 2019. Since its web design new designs, layouts and icons keep updating. Took a lot of time to complete it. So this is an ongoing project. I am the only one involved in building it. So eventually it took a lot of time to build it because all the documentation, research, and learning process was done by me. So I will be regularly updating the website with extended features.

How

As said earlier I am building this website from scratch. I am using HTML5 to build the front-end, CSS4 for the styling, Bootstrap4 for the responsiveness of the website and Javascript for the animations. I am using Brackets IDE to type my code. This is an amazing IDE with extended features like “live preview” on a browser, auto code completion, syntax highlighting, and many more. I have also referred to some concepts and code snippets from unique sources. Refer the RAW code for more details.
